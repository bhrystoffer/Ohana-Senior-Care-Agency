12 — Interactions

Motion & Interaction System

Transition Durations

fast — 150ms (micro interactions, hover states)
base — 240ms (buttons, nav links, card borders)
slow — 360ms (reveals, drawer open)
slower — 520ms (page transitions, hero entrances)

Easing Curves

ease-out

cubic-bezier(0, 0, 0.2, 1) — starts fast, decelerates. Used for entering elements.

ease-in-out

cubic-bezier(0.4, 0, 0.2, 1) — gradual start and end. Used for drawers and panels.

ease-spring

cubic-bezier(0.34, 1.56, 0.64, 1) — slight overshoot. Used sparingly for delight moments.

Scroll Reveal

Elements with .reveal start at opacity:0, translateY(20px) and transition to visible when entering the viewport via IntersectionObserver. Add .stagger to a parent for child delay steps of 60ms each.

Global Rules

  • Cards: translateY(-4px) + green shadow on hover
  • Buttons: translateY(-1px) + shadow on hover
  • Nav: compresses height on scroll (80→62px)
  • Text links: arrow shifts translateX(4px)
  • Logo carousel: auto-scroll pauses on hover
  • No animation for prefers-reduced-motion
